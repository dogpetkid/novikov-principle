Quick start

This is brief section in order to help you understand the game and it's menus quickly.

The title screen has 4 buttons:

  • Play: Brings you to where you last were playing or starts the game on first launch.
  • Level Select: Brings you to a menu with sections you can jump to.
  • Options: Contains the help section, options for you to change, and allow you delete save data.
  • Exit: Exit the game... (useful for fullscreen).

The basic controls are in game:

  • WASD / Arrow keys: to move.
  • Space: "jump".
  • Zero (0): restart room.
  • ESC: Go to menu.
  • F4: Fullscreen (works only in game).
